Design a homepage concept for "Nech Code", a founder-led technology partner for businesses and organizations in Indonesia.

- Nech Code builds practical digital solutions
- Services include websites, landing pages, internal systems, automation, integrations, and AI-assisted digital workflows
- The brand should feel modern, credible, practical, and founder-led
- It should not feel like a generic SaaS startup
- It should not feel like a corporate enterprise website
- It should not feel like a personal portfolio

Visual identity:
- Primary deep blue: #1D5A8D
- Accent cyan: #00BCD4
- Warm brown accent: #7E4B00
- Base cream background: #F5EEDC

Art direction:
- clean warm-tech studio aesthetic
- editorial composition
- modular layout
- asymmetric hierarchy
- premium whitespace
- soft shadows
- rounded surfaces
- lightweight, believable, not AI-looking

Hero inspiration blend:
- base layout logic from a monochrome editorial launch page
- supporting preview block inspired by a technology showcase wall
- spacing and clarity inspired by a clean UI component landing page
- strong restrained typography inspired by minimal launch heroes

Homepage goal:
- explain clearly what Nech Code does
- build trust
- feel distinct and art-directed
- make visitors want to consult or view selected work

Use Indonesian copy for the visible content.
